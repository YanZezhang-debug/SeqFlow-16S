## [1.0.0] - 2025-11-01

### 新增功能
- ✨ **智能文件名识别系统**
  - 支持 6 种常见的文件命名格式
  - 可通过配置文件自定义命名规则
  - 自动识别正反向测序文件配对
  
- ✨ **完整的分析流程**
  - 序列质量控制和评估
  - 基于图论的序列组装算法
  - NCBI BLAST 在线物种鉴定
  - 自动生成 Excel 和文本格式报告
  
- ✨ **BLAST 断点续传功能**
  - 自动保存分析进度
  - 中断后可继续执行
  - 支持多种 BLAST 数据库选择
  
- ✨ **双模式操作**
  - 交互式向导模式（适合新手）
  - 批处理自动化模式（适合批量处理）
  
- ✨ **灵活的配置系统**
  - YAML 格式配置文件
  - 可自定义所有分析参数
  - 支持添加自定义文件名规则

### 核心模块
- `filename_parser.py` - 文件名解析和样本识别
- `graph_based_assembly.py` - 序列组装引擎
- `ncbi_blast_analysis.py` - BLAST 分析模块
- `sanger_16s_pipeline.py` - 完整流程管理器

### 文档
- 完整的 README 使用文档
- 详细的配置文件说明
- 常见问题解答
- 多种使用场景示例
